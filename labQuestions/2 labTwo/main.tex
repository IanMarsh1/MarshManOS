\section{How is your console like the ancient TTY subsystem in Unix as described?}
The console we are creating that runs in the browser is functionally similar to the TTY subsystem. The first similarity is the way backspaces are done. As someone who makes A LOT of mistakes typing, trying to work in a world where everything I type is set in stone would cut my life expectancy in half. Another similarity is the use of drivers. Right now, the only thing we have a driver for is the keyboard that handles translation of key stokes, but if it were to expand, we would see many of the same drivers. The TTY subsystem is similar to the web-based one we are creating today, although the implementation is decades after.
